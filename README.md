# 🏦 Gringotts Wizarding Bank

A full-stack digital banking application inspired by the magical world of Harry Potter. Gringotts Wizarding Bank provides a secure and modern banking experience with OTP-based authentication, digital vault management, transaction processing, downloadable PDF receipts, and an elegant animated user interface.

---

## 📖 Overview

Gringotts Wizarding Bank is a feature-rich banking application developed to simulate the core functionalities of a modern digital bank while incorporating a magical wizarding theme. The project demonstrates full-stack web development, secure authentication, RESTful APIs, database management, responsive UI design, and professional software architecture.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Themonsterk9/gringotts-wizarding-bank.git
```

### Install Dependencies

#### Client

```bash
cd client
npm install
```

#### Server

```bash
cd server
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the `server` folder.

```env
PORT=5001

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

EMAIL_USER=your_email@gmail.com

EMAIL_PASS=your_gmail_app_password
```

---

## ▶️ Run the Project

### Backend

```bash
cd server
npm run dev
```

### Frontend

```bash
cd client
npm run dev
```
